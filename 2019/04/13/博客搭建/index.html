博客搭建

GIT博客搭建步骤
T1. 安装node.js 并配置环境变量。
安装好后 打开cmd，输入命令检查是否成功：node -v 和 npm -v

T2.安装 git 并配置环境变量 。
可以选择适合自己的操作系统：https://git-scm.com/downloads

T3. 注册github，新建项目并配置SSH 和github的SSH keys
NO.1 右上角 your repositories，点击 New. 要注意三点：1.命名要以自己的账户名开头，比如：00cream00.github.io 。 2Repository name 要以 .github.io 结尾。3.勾选Initialize this repository with a README
NO.2 配置SSH 并配置github ssh，点击进入git安装目录，点击 git-bush.exe 输入：ssh-keygen -t -C “user.email” ,一直回车，碰到选择就选 y 直到出现秘钥图像
NO.3 右上角 setting –> SSH and GPG keys –> SSH keys 选择 New SSH key ，随便输入名字，在输入ssh秘钥 ，位于本机的 C:\Users\我的.ssh 里的 id_rsa.pub
T4. 安装hexo
NO.1 创建目录 blog 如： D：\blog ,右击 用git bush here打开，
NO.2 开始安装Hexo，输入安装命令 npm install hexo -g 。 感觉报错就输入：cnpm install hexo -g，下面涉及npm命令的可能都要用cnpm代替了。
NO.3 输入hexo -v，检查hexo是否安装成功，
NO.4 输入hexo init 123Blog，初始化该文件夹并输入博客名，（有点漫长的等待。。。）
NO.5 cd 123Blog 进入博客名目录下
NO.6 输入cnpm install，安装所需要的组件，
NO.7 输入hexo s -p 5555 ，开启服务器，访问该网址 localhost：5555，（默认端口是4000）正式体验Hexo （这是本地的）
T5. 安装sublime
T6. hexo 发布至 github
打开 sublime 将 创建的 123Blog 拖进去，打开 source–>_posts –> hello world,可以修改博客名。 打开 —_config.yml,主要修改两个地方 url:https://username.github.io.git
deploy:
type: git
repo: https://git@github.com/username/username.github.io.git （repo值是你在github项目里的ssh（右下角））
branch: master

作者也可以修改
T7.
cnpm install hexo-deployer-git –save
hexo g
hexo d 这时会跳出登录弹框
没有提示要输入：git config –global user.name “name”
git config –global user.email “@qq.com”
就自己输入，将Tep.7 再重来一遍
好了就访问下 localhost：4000/自己设定的端口
T8.
启动hexo服务：进去 C:\blog ，右击123Blog。用 git bush打开，输入：hexo server
打开浏览器输： localhost：4000/自己设定的端口 到本地hexo
打开线上博客：https://00cream00.github.io/
本地编写：F:\hexo\blog\source_posts，直接拖到sublime
上传到线上博客： hexo d -g
