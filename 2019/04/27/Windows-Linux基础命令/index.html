    Linux权限**

    Ctrl+C 退出当前命令

    权限
    第一列中共有10位
    第一位表示文件类型：

    d 这是目录
l 链接文件
- 二进制文件（文本文件）

    第2-4位表示当前文件所有者的权限 r（读）4 w（写）2 x（执行）1
    第5-7位表示当前文件所有组的权限 r-x
    第8-10表示其他人对文件的权限 r-x
    例：

    l --x  r-x --x    151
   链接文件，所有者的权限-执行， 所有组的权限--读取和执行，其他人的-执行
按照读写执行 固定顺序，有间隔就用-   最高7=4+2+1：
      341  -wx  r--  --x   
      756  rwx  r-x rw-
