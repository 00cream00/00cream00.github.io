Windows,Linux基础命令

Apr 27, 2019

Windows
命令都在终端执行

1.ping : 检查网络连通性 ping url=ping /？
start www.baidu.com 打开百度
2.ipconfig ： 查看IP
ipconfig /? 查看有什么命令
ipconfig /all ：可看物理地址 ipconfig下所有的网卡信息
换ip：
ipconfig/release ：释放ip
ipconfig/renew ：重新获取IP
ipconfig /displayds 查看本机dns ipconfig/ flushdns 清空dns

  1. arp -a查看本机ARP表，通过网络地址找到物理地址 ，可以形成双向欺骗
    route print :查看本机路由表

4.netstat -ano :查看本机开启的端口
netstat -ano|findstr 80 :查看具体的某个端口,管道符：前面命令输出 的结果当做后面命令的开始， 查看端口
5.tasklist ：查看本机所有任务列表（ tasklist /? 不知道具体命令，调用全部相关命令）
tasklist |findstr 9268： 获取具体某个任务列表 找本机9268这
tasklill /f /im qq.exe 结束程序进程

  1. dir ：列举目录，文件夹。【 Linux 里列举目录是 ls 】（cd dur交互，明白含义） 
    dir c:\  查看c盘     dir /a  查看隐藏文件
    cd : 切换目录文件夹 ，在目录里可以创建文件或文件夹。 比如用mkdir创建的目录（加后那是文件）
    cd .. :返回上层目录
    cd \ 或者 cd / ：回到根目录
  2. cls :清屏
  3. Ctrl+C :停止命令
  4. Ctrl+C :停止命令
    9.whoami :查看当前用户
  5. hostname : 查看当前计算机命
  6. 删除文件夹和文件
    del 文件 /文件夹 用来删除文件和清空文件夹， rd 文件夹 用来删除空文件夹
    补充：如果不是空文件夹，就先del 文件夹 先删掉文件夹里的内容，再 rd 文件夹。

12systeminfo :获取系统详细信息
services.msc 查看本地服务
net view 查看局域网的其他主机

13.path /set :系统环境变量
变量：此电脑 -属性- 高级系统设置 - 环境 变量（path最后面加 ‘’；’）

14 color ：改变cmd颜色
pid 进程ID号，表示在系统中命令进程

创建文件，写入，读取

  1. mkdir test/test.txt : 创建文件夹/文件 （这个是目录，不要加后缀） 也可以创建文件，但是echo和type就不能作用这个文件，必须把后缀去掉件中写数据

  2. echo 123 >test.txt :创建文件并写入（这是个文件，要加后缀，是mkdir创建的目录里的文件）
    md 123 创建文件123 ，del 123 删除文件123

  3. type test.txt ： 读取文件： notepad test.txt 用notepad打开文件
  4. copy test.xt d:\ 复制文件到d盘
    ren 1.txt 2.txt 把文件名1改成 2

Windows用户管理*
net user

  1. net user :查看系统用户
  2. net user username :查看具体某个用户信息
  3. net user hack$ 123 /add :添加用户和密码
    （用户名后加上$可以隐藏创建的账号，默认user组，权限低）
    4 net user hack 345 :修改密码

  4. net user hack /del : 删除用户

    net localgroup
    1.net localgroup :查看本地组
    关注 administrators /user/Remote Desktop Users
    2 net localgroup test /add :创建组（ 此处test随意取名）

  5. net localgroup administrators hack /add :往管理员组添加用户
    4.net localgroup administrators test /del ：往管理员组删除用户

5.创建隐藏用户：
net user admin$ 123.coom /add

注意：添加删除用户都需要管理员权限

IP：表示网络节点（设备）在整个网络中的位置，使 用点分十进制表示 x的范围是0~255
以127开头的ip，127.x.x.x 都是本地
本地：127.0.0.1 ，网关：192.168.179.2，子网掩码：255.255.255.0

ping 127.0.0.0   .255表示广播地址，0表示
ping  127.0.0.1  测试本地网卡，本地回环地址

ping 192.168.11.1 -t  重复ping
ping 127.0.0.1 | ipconfig  “”管道符“” 前    面会执行但不会显示，只显示后面  （ipconfig）

.路径
相对路径：相对某个文件或者文件夹 .\ 当前目录 ..\ 表示上级目录
绝对路径：完整的，从根目录写起的路径

以点开头的文件默认情况下是隐藏文件
